# Mental Arithmetic

A simple Java Swing application for practicing mental arithmetic. The application presents multiplication and division problems and tracks the user's score.

## Building the Project

This is a NetBeans project that uses Ant for its build system. To build the project, you can use the `ant` command in the root directory of the project:

```bash
ant
```

This will create a `dist` directory containing the executable JAR file.

## Running the Application

After building the project, you can run the application by executing the JAR file:

```bash
java -jar dist/MentalArithmetic.jar
```
